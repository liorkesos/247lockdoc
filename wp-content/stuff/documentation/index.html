Installation - top

Permatex requires WordPress v3.5 or higher to function properly!

Do not upload the ZIP file you downloaded from ThemeForest. Unpack it first on your computer.

Permatex can easily be installed as any other standard WordPress theme using one of 3 ways:

  • Upload "Permatex" folder to your /wp-content/themes/ folder and then activate it under Appearance, or
  • Go to Appearance - Install Themes - Upload and point to "permatex.zip" file in order to install the theme, or
  • Use Envato WordPress Toolkit

After installing a message will be displayed warning you to import the demo content. You don't have to do it but it will help you setup the theme.

All theme configuration is done trough the Theme Customizer and everything else is standard WordPress stuff.

Updating the theme

Latest version of the theme can always be downloaded from ThemeForest, on your profile under the "Downloads" tab. Download the new version from there, unpack it on your computer and connect via FTP to your server. Find /wp-content/themes/permatex/ folder and replace all files in it with new files found in the /permatex/ folder in ZIP file.
By doing this you will not lose any content or settings. You will lose all file modifications.

Choosing the type of front page - top

We strongly recommend importing the demo content under Appearance - Permatex. You don't have to do it but it will help you setup the theme withing minutes and set all the options for you.

Open WP admin and go to Settings - Reading. First option "Your latest posts" will display latest posts on the front page. You can still have the landing page but on some other URL. IE "/landing-page/" by creating a page titled "Landing Page" and giving it the "Front Page" template. Please note that you can only have one instance of the landing page.

Other option "A static page" will put the landing page on the front page, while the posts (blog) will be available on some other URL. Create two pages; name one "Blog" and the other one "Front Page" and give it the "Front Page" template. Don't put any content in those pages. Select pages "Front Page" and "Blog", respectively from two dropdowns. Blog will be available on "/blog/" and will show posts from all categories. If you used the "demo data import" function everything will already be preset for you using this method.

Front Page - top

Permatex features an extremely versatile front-page that can easily be customized for many uses. Please follow the instructions below to make the customization as fast as possible.

Header
Header logo icon and logo image as well as the color are set universaly for the whole site. The menu is however configured separately for the front page (Front Page options section) and all other pages (Header options section).

Front Page Header section
There are 2 types of front page headers: contact/leads form and slider. You can also completely remove the header section.
Slider shows posts from a category selected in Theme Customizer - Front Page - Slider Posts Category setting. All other settings are configured in the theme customizer as well. When editing posts you'll see a new box labeled "Slide options". Use the box to set an optional slide link. Slider is limited to a maximum of 20 slides and they are ordered chronologically from the most recent one. Image is of course set as the featured image of the slide/post. If the post doesn't have a featured image it's ignored as far as the slider goes.

Content / Sections
First section on the front page is a sidebar area called "First Section on Front Page" and it looks best if you use "Text With Icon" widget in it. If you leave the sidebar empty that section is going to be hidden. Sections are pages which have the Front Page Section template set. To add a new section create a new page, put any content in it and be sure to set the "Front Page Section" template. Most sections just use one of the shortcodes to show content such as screenshots or Twitter feed. See demo pages' content for examples. When editing pages you'll see a "Front Page Section Options" box that allows you to set additional options for each individual section/page.
On the pages list in WordPress admin you can easily drag&drop pages to reorder them (cursor changes to four-directional when you hover over page rows). Be sure to click on the "Sort by Order" link above the pages table to view them in the right order.

Footer
Footer copyright line, logo and "to top" button are configured in Theme Customizer - Footer. Those settings are global for the whole site.

Shortcodes - top

Permatex comes with a set of easy to use shortcodes described below. Besides these shortcodes you are of course welcome to use any other custom shortcodes that are compatible with WP v3.5. The "Shortcodes" page in the demo shows practical usage of all shortcodes.

[button href="#" target="" size="" color="" class=""]button text[/button]

Displays a link styled as a button.
Shortcode content sets the button text. Href, class and target attributes are equivalent to the same attributes in an A tag. Following sizes are available: mini, small, large and default (leave attribute blank); and these colors as well: warning, danger, success, info, inverse and default (leave attribute blank).

[border /]

Displays a simple horizontal border.
This shortcode has no options.

[center]any content[/center]

Centers the content in it.
This shortcode has no options.

[clear]

It's used to clear floating after elements. Use it only if there are elements that don't float properly by placing clear before/after them.
This shortcode has no options.

[download_button href="#" target="" title="Download now" type="apple" class="" /]

Displays a download button with Play store or App store.
Href, class, title and target attributes are equivalent to the same attributes in an A tag. Type can be set to android or apple.

[faq q="Question"]Answer[/faq]

Displays a question with a collapsible answer.
Question (q) is the only option.

[font_icon]

Displays one of +300 FontAwesome font icons.
Shortcode content is the icon name. You can find a list of all icon names on the FontAwesome site. "size" attribute controls the icon size; following sizes are available: "2x", "3x" and "4x". "class" parameter can take any CSS clase name if you want to additionaly style the icon.

[fp_widgets /]

Displays the front page sidebar.
This shortcode has no options.

[gallery]

Permatex uses a modified native WP Gallery shortcode. By default images are taken from the gallery of the post/page the shortcode is used on but that can be changed using the "id" attribute. For a full list of attributes please visit the Codex page.

[gmap zoom="12" height="400" bubble="true"]address[/gmap]

Displays a Google map for the defined address.
Shortcode content sets the address. Zoom should be an integer between 1 and 14. 14 being the most zoomed-in setting (street level). Width is always 100% (use other shortcodes to get half, third ..) while height needs to be defined in pixels. Bubble can be set to "true" or "false" and defines whether the address bubble above the address marker is visible or not.

[half]content[/half] [half_last]content[/half_last]

Use this shortcode to create a 50% wide content box. Please mind the "regular" and last box. You have to use an appropriate shortcode.
This shortcode has no attributes.

[newsletter default_name="Enter you name" default_email="Enter your email" button="Subscribe now" /]

Displays a newsletter signup form powered by MailChimp.

Listed options control the default value of input fields.
MailChimp API key and list name are set in Theme Customizer options under the Newsletter section.
Please use this shortcode only on the front page.

[quarter]content[/quarter] [quarter]content[/quarter] [quarter]content[/quarter] [quarter_last]content[/quarter_last]

Use this shortcode to create a 25% wide content box. Please mind the "regular" and last boxes. You have to use an appropriate shortcode.
This shortcode has no attributes.

[quote smallprint="quote author" quotes="1"]quote text[/quote]

Displays a quote formatted text along with an optional quote author in small print.
"smallprint" attribute is optional. "quotes" attribute defines if the text should be wrapped in fancy quotes or not.

[quoterotate][quote]quote #1[/quote][quote]quote #2[/quote][/quoterotate]

Rotates multiple quotes.
This shortcode has no options. Please see quote shortcode for its options. This shortcode can only be used on the front page.

[sc]some shortcode that should be displayed unparsed[/sc]

Used only for documentation purposes. Enables you to display a shortcode in raw, unparsed state.
This shortcode has no options.

[screenshots post_id="" /]

Displays screenshots in form of a carousel.
By default images attached to the current page are used but a custom page can be defined with the post_id paramater.
This shortcode can only be used on the front page.

[social_icon href="#" target="" color="" class="" tooltip=""]icon[/social_icon]

Displays a large font icon with tooltip.
Shortcode content sets the icon. Full list can be found on Font Awesome Cheatsheet. Href, class, target and color attributes are equivalent to the same attributes in an A tag. Tooltip sets the fancy hoverable tooltip text.

[spacer]

Adds a vertical spacer.
This shortcode has no options.

[superlist title="" icon=""]text[/superlist]

Displays a fancy list item with icon, title and text.
Shortcode content sets the small text. Title parameter the large text and icon the icon. Full list of icons can be found on Font Awesome Cheatsheet. This shortcode can only be used on the front page.

[tabgroup][tab title="tab 1"]content[/tab][tab title="tab 2"]content[/tab][/tabgroup]

Creates a group of tabs. Each tab has a title and content. Please note the syntax - a shortcode within a shortcode has to be used.
Title attribute sets the individual tab's title.

[title size="4"]content[/title]

Displays a title in a heading tag of the set size.
Size attribute can be set to: 3, 4, 5 or 6.

[third]content[/third] [third]content[/third] [third_last]content[/third_last]

Use this shortcode to create a 33% wide content box. Please mind the "regular" and last boxes. You have to use an appropriate shortcode.
This shortcode has no attributes.

[twitter count="1" avatar_size="32"]username[/twitter]

Displays the latest tweet from the specified Twitter username.
Shortcode content defines the Twitter username. Parameters "count" and "avatar_size" control the number of tweets displayed and the site of avatar image in pixels.
This shortcode can only be used on the front page.

SEO - top

SEO options contain the following three meta tags - title, description and keywords. Options for the front page and default options are set in the Theme Customizer under the Title, Tagline & SEO section.

Options for individual pages and posts are set in the "SEO Options" box available in the individual post/page edit screen. For posts that don't have custom values defined default ones are used. To ease the editing we've provided the following variables that can be used when writing the meta tags for individual posts: {site-title}, {site-tagline}, {site-description}, {site-keywords} and {page-title}. Variables use the default SEO values set in Theme Customizer.

Theme Tools - top

Navigate to Appearance - Permatex Tools to access theme tools. The following sections are available;

Import Demo Data

The easiest way to get your site going is to import the demo data. It will make the site look exactly like the demo site and enable you to easily change anything you need instead of creating content from scratch.
No data that's already on the site will be deleted or altered.

Theme Options

This is just a link to remind you that all theme options are configured via the Theme Customizer.

Export Options

If you're moving the site to another WP installation this is the easiest way to transfer Permatex options. Please note that this only exports Permatex options, not any general WP options.
Either download the export file or copy the provided data.

Import Options

After you've exported the options import them either by uploading the export file or by pasting the export data into the text area.

Reset Options

If you feel something's broken resetting theme's options to default values might help fix the problem.
There's no undo button, so be carefull.
